# FitLog

PulseLog is a personal fitness tracker built for one user.

Main highlight: **vibe coded with Cursor**.

I built this because I wanted a clean place to track training and daily health metrics, and then quickly analyze progress over time.

## What this project does

- Tracks daily training notes (training summary)
- Tracks daily metrics (sleep, steps, protein, calories, body weight)
- Shows a yearly workout consistency heatmap
- Shows monthly analysis charts for metrics
- Keeps access simple with a single-user login

## Product vibe

- Minimal and dark-first design
- Fast and focused (no unnecessary complexity)
- Mobile friendly
- Built for consistency, not clutter

## Who it is for

Right now this project is for a **single user** setup (me).  
It can be expanded later if needed, but the current goal is simple personal tracking + analysis.

## Core flow

1. Open Home to view motivation, yearly consistency, and monthly analytics.
2. Go to Logs and sign in.
3. Select a date, add training summary + daily metrics.
4. Come back to Home and watch progress reflect in charts/heatmap.

## Note

This app is intentionally opinionated and lightweight.  
It does one thing well: **help track and analyze fitness consistency over time**.
